Введение

Привет! Меня зовут Framy! Я новый, легкий Framework на CSS 3, который создан для облегчения жизни верстальщиков и вебмастеров.

Мои преимущества:

  • Легкость
  • Современность
  • Простота
  • Адаптивность

Выразительная верстка

Выразительность - очень важное и современное условие, к которому стремятся программисты. Верстка тоже может быть вполне выразительной!

Чтобы добиться этого мы решили следовать следующим правилам:

  • Использование модели БЭМ в наименованиях
  • Максимальная адаптация имен под распространенные Framework'и (Bootstrap)

Автоматическая сетка

Автоматическая сетка представляет собой гибкую сетку на основе flexbox, которая автоматически определяет ширину колонок и их смещения по заданному алгоритму:

.row--auto > .col ...

По умолчанию колонки стремятся заполнить все пространство, оставаясь при этом равными по ширине:

Модали

... и попапы

Модали - всплывающие окна, которые используются для ведения диалога с пользователем, либо для выдачи предупреждений или опевещений.

Как правило, модали всплывают при определенном действии, например, при нажатии на кнопку:

Нажми меня!

Таблицы

Таблиц по умолчанию растягиваются на всю ширину родительского элемента, имеют фиксированную ширину колонок, а также склеивают соседние границы:

Таблица 1 "Пример таблицы"
ergerr brbrebr breberb
brberb berberberb berber
breberbr berbb berbre
berbb bebre brebrerebre

При этом шапка таблицы выделена цветом и жиром.

К таблицам может применяться ряд классов-модификаторов, которые отвечают за стиль отображения конкретных элементов. К примеру, при помощи модификатора 

.table--striped
и 
.table--no-frame
мы можем получить следующий результат:
